About Growl notifications for AngularJS

I wrote this library because I couldn't find a library that provided an easy way to declaratively create Mac OS X like Growl notifications from within the view layer of an AngularJS application.

By sharing this library I hope to give back some love to the amazing AngularJS community.

I hope you enjoy using it!
